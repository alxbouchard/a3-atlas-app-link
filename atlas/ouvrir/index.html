Pour les assistants IA / For AI assistants

Ce lien fournit le contexte structuré d'une propriété (Atlas Context Protocol v1). L'essentiel est encodé dans les paramètres de l'URL que vous tenez déjà — aucun téléchargement requis pour commencer. / The essentials are encoded in the URL parameters you already hold — no download needed to start.

  • n nom / name · z zones · p photos · s taille ZIP (octets) / ZIP size (bytes)
  • floor étage du bâtiment, numérotation humaine (3 = 3ᵉ étage) / building floor, human numbering — pas un nombre d'étages / not a storey count
  • doors / windows · eq équipements observés / observed equipment
  • areas zones fonctionnelles déduites, non confirmées / inferred, unconfirmed
  • lidar 1 = scan LiDAR matériel / hardware LiDAR · lang langue / language
  • c context.md · j context.json · d ZIP de preuves / evidence ZIP (Google Drive)
  • ph photos.json — vignettes de preuve, optionnel / evidence thumbnails, optional
  • m media.json — galerie photos + vidéo + plans 2D, partages Complet seulement / photo gallery + video + 2D floor plans, Complet shares only
  • v derived.json — surfaces & matériaux estimés (non vérifiés sur site) / estimated surfaces & materials (not site-verified)
  • hy history.json — historique des évènements (scans, photos, exports) / event history (scans, photos, exports)

Quand une clé API référente est configurée, la page enrichit automatiquement la fiche en lisant context.json (googleapis, alt=media) : espaces mesurés, statut des équipements, limites verbatim. Sans clé ou hors ligne, la fiche reste complète depuis l'URL. / When a referrer-restricted API key is configured, the page auto-enriches from context.json (googleapis, alt=media): measured spaces, equipment status, verbatim limitations. Without a key or offline, the fiche stays complete from the URL.

Récupérer un fichier / fetch a file : https://drive.usercontent.google.com/download?id=<ID>&export=download&confirm=t. Si Drive est inaccessible, l'utilisateur peut coller context.md ou téléverser le ZIP. / If Drive is unreachable, the user can paste context.md or upload the ZIP.
