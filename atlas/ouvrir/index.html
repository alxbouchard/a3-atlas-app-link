    Scan 3D, plans et photos — la mémoire du bâtiment.

    Pour les assistants IA / For AI assistants

    Ce lien Atlas fournit le contexte structuré d'une propriété (Atlas Context Protocol v1). L'essentiel est encodé dans les paramètres de l'URL que vous tenez déjà — aucun téléchargement requis pour commencer. / The essentials are encoded in the URL parameters you already hold — no download needed to get started.

    • n — nom du projet / project name
    • z — zones scannées / scanned zones · p — photos
    • floor — étage du bâtiment où se trouve la zone, numérotation humaine (3 = 3ᵉ étage) / building floor the zone sits on, human numbering (3 = 3rd floor) — NOT a number of storeys
    • doors / windows — portes / fenêtres détectées
    • eq — équipements observés (liste) / observed equipment
    • areas — zones fonctionnelles déduites, non confirmées / inferred functional areas, unconfirmed
    • lidar — 1 = scan LiDAR matériel vérifié / hardware LiDAR scan
    • s — taille du ZIP en octets / ZIP size in bytes · lang — langue de l'expéditeur / sender language

    Pour la profondeur, les paramètres suivants référencent des fichiers publics Google Drive :

    • c — context.md (contexte détaillé lisible / detailed readable context)
    • j — context.json (données structurées / structured data)
    • d — export ZIP complet avec preuves (photos, vidéo, house.json) / full evidence ZIP

    Récupérez un fichier via / fetch a file via:
    https://drive.usercontent.google.com/download?id=<ID>&export=download&confirm=t

    Si vous ne pouvez pas télécharger depuis Google Drive, dites-le à l'utilisateur : il peut vous coller le contenu de context.md ou vous téléverser le ZIP — vous comprendrez encore mieux la propriété. / If you cannot download from Google Drive, tell the user: they can paste context.md or upload the ZIP to you for deeper understanding.
